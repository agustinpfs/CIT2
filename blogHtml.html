¿Qué es HTML? Concepto, estructura, principales etiquetas y ejemplos.

13/7/20 por agustinpfs

HTML

HTML5 hace referencia a la última versión de HTML.

“HyperText Markup Language“ (Lenguaje de Marcado de Hipertextos). “Hipertexto" se refiere a los enlaces que conectan las páginas web, aspecto fundamental de la web. Son la pieza clave de la navegación web.

Estos archivos llevan la extensión .html y son archivos que pueden ser interpretados por el navegador. Con solo abrir el bloc de notas en Windows o el TextEdit en Mac, escribir “hola” y guardarlo con extensión .html, será suficiente para que ese archivo pueda ser abierto y mostrar nuestro “hola” en la ventana del navegador (Chrome, Firefox, etc).

Sin embargo, para crear un documento html de la manera correcta deberá tener cierta estructura que veremos en este artículo. Además, en vez de usar el bloc de notas, usaremos un editor de código que nos permite escribir estos archivos de manera más cómoda, mas rápida y con menos errores.

Elegiremos Visual Studio Code por su potencia, fácil instalación y por ser uno de los editores de código más populares en este momento. Puedes descargarlo desde su página oficial.

HTML es el archivo más básico de una página web y en el definiremos su estructura y alojaremos la información que contendrá (texto, enlaces, imágenes, vídeos, audios, etc).

Para ello utilizaremos las etiquetas (tags) que se encargan de clasificar y guardar la información en un archivo HTML. Por lo general los elementos HTML (tags) se determinan como pares de etiquetas (apertura y cierre). Se definen entre los signos “menor que” (<) y “mayor que” (>). La etiqueta de cierre lleva una barra antes del nombre:
Apertura -> <nombre-etiqueta>
Cierre -> </nombre-etiqueta> (algunos elementos no llevan etiqueta de cierre)

Atributos

Los atributos proporcionan información adicional sobre las etiquetas HTML. Se colocan en las etiquetas de apertura y son de la forma atributo=”valor”.

Ejemplos:

<a href="https://www.google.com">Link a Google</a>

La etiqueta <a> representa un enlace y el atributo href la ruta hacia donde se dirige.

<img src="ruta" alt="descripcion">

La etiqueta <img> representa una imagen, el atributo src(fuente) la ruta en donde se aloja la imagen y el atributo alt la descripción de la imagen (se muestra si la imagen no carga y de utilidad para los lectores de pantalla).

Etiquetas semánticas

  • encabezado <header></header>
  • menú de navegación <nav></nav>
  • contenido principal <main></main>
  • articulo <article></article>
  • sección <section></section>
  • sidebar (barra lateral) <aside></aside>
  • pie de página <footer></footer>


Estas etiquetas son de carácter semántico, el comportamiento del contenido de estas etiquetas no varía, es decir un html escrito con estas etiquetas tiene el mismo comportamiento si todas las etiquetas fueran “divs” <div>.

Pero utilizar estas etiquetas ayuda a la legibilidad del código, a la accesibilidad para los usuarios con limitaciones visuales (lectores de pantalla) y pueden incidir en los resultados de búsqueda por parte de los motores de búsqueda.
