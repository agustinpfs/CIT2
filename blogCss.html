¿Qué es CSS? Definición, sintaxis, selectores y propiedades.

15/7/20 por agustinpfs
Ver video tutorial de este artículo

CSS

CSS son las iniciales de “Cascading Style Sheets” (Hojas de Estilo en Cascada).

"Cascada" se refiere a cierto orden de prioridad en que se aplicarán los estilos para un mismo elemento. Este concepto se comprenderá mejor a medida que se avanze en el aprendizaje.

Con Html creamos elementos en los cuales colocamos información (texto, imágenes, enlaces, etc). Estos elementos conforman las partes o secciones de nuestra página web.

Con CSS podremos manipular estas secciones creadas en el HTML, dándoles posiciones específicas a cada una de ellas.

También nos permitirá definir colores, tipo y tamaño de fuente, espacios, etc. Css es el encargado del orden y estética para que un sitio web se vea según lo planeado.

Selectores

Son referencias tomadas de los elementos de la página .html para poder aplicarles estilos a dichos elementos.

Los más usados son:

  • selector de etiqueta
  • selector de clase

Selector de etiqueta.

Toma dierectamente el nombre de la etiqueta y le aplica los estilos.

Ejemplo

Selector de clase.

Una clase(class) es un atributo(*) de tipo class="nombreclase". En este caso "nombreclase" se utilizará como selector en la hoja de estilos css anteponiendo un punto. Al mismo atributo class se le pueden aplicar varias clases separadas por un espacio class="unaclase otraclase". Una misma clase puede aplicarse a varios elementos. Es la manera mas usual de aplicar estilos.

(*)Los atributos proporcionan información adicional sobre las etiquetas HTML.

Propiedades CSS

Dicta qué aspecto del selector se cambiará visualmente. El valor es a lo que cambiará la propiedad CSS elegida

h1 {color: blue}

Una propiedad (color) dice que aspecto del selector (h1) cambiará visualmente. El valor (blue) es a lo que cambiará la propiedad.

Propiedades mas usuales:

font-family tipo de letra (fuente)

font-size tamaño de letra

color color del texto

width anchura

height altura

padding margen interior

margin margen exterior

border color, ancho y estilo del borde

background-color color del fondo del elemento

background color, imagen, position

text-align alineación del texto

display tipo de caja(*)

(*)Especifica el tipo de caja que se aplica a un elemento. Que comportamiento va a tener.

Algunos de los valores de la propiedad "display" son:

  • none: hace que no se genere caja
  • block: caja rectangular que forma un bloque
  • inline-block: caja en-línea, pero que se formatea como una caja de tipo block
  • inline: caja en-línea y está incluida en una caja de tipo block

Box Model

En CSS, el término “Box Model” (modelo de caja) es esencialmente una caja que envuelve todos los elementos HTML.

CSS interpreta a cada elemento html, es decir, a todo elemento definido entre una etiqueta de apertura y una etiqueta de cierre como una caja rectangular definida por un borde, un margen interno, un margen externo y el contenido que encierra.

El tamaño del contenido puede darse por lo que ocupe la información que contiene el elemento o por un ancho y una altura en unidades de medida aplicado con css.

Entonces las propiedades con las que podemos definir los tamaños del borde, margen, margen interno y contenido son: Para los márgenes y bordes: border, margin, padding. Para el contenido: width(anchura) y height(altura). Si no se especifican estas propiedades el tamaño lo determina el propio contenido.

También cabe aclarar que los elementos definidos como elementos “en línea” no le podemos aplicar una altura y un ancho. Antes habría que convertirlos en elementos de “bloque” por medio de la propiedad “display”.

Box Model


Si el tamaño del contenido de la caja está determinado por las propiedades height y width podemos usar la propiedad box-sizing.
box-sizing: content-box (valor por defecto) determina que al alto y al ancho definido del elemento siempre indicará el tamaño del contenido sin tener en cuenta los tamaños del padding y del border. Es decir, el tamaño de la caja será igual al ancho(width) y alto(height) mas los tamaños del border y padding.


En cambio, con box-sizing: border-box determina que al alto y al ancho definido del elemento incluirá los tamaños del padding y del border. Es decir el tamaño de la caja será igual al ancho(width) y alto(height) definido en el css.
