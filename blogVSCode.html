Visual Studio Code. 22 atajos y utilidades para simplificarte la vida.

15/5/20 por agustinpfs

1- Creación estructura Html:

! Enter ó html:5 Enter

2- Crear etiqueta con el nombre de la clase:

etiqueta.nombreclase Enter

3- Duplicar línea o bloque.

Colocar cursor sobre la línea a duplicar o seleccionar el bloque de código a duplicar.

alt shift flecha arriba ó abajo

4- Mover línea o bloque

Colocar cursor sobre la línea a mover o seleccionar el bloque de código a mover.

alt flecha arriba ó abajo

5- Múltiple cursor

alt click

6- Múltiple cursor en columna

Windows:

ctrl Alt flecha arriba o abajo

macOS:

cmd Alt flecha arriba o abajo

7- Seleccionar línea entera

Windows:

ctrl L(minúscula)

macOS:

cmd L(minúscula)

8- Deseleccionar

Cuando seleccionamos varias líneas con cmd L es posible que nos pasemos.

Para deseleccionar:

Shift flecha arriba

9- Buscar repeticiones de una selección

cmd d

10- Copiar o cortar línea entera

Sólo basta con colocar el cursor en cualquier parte de la línea

Windows:

ctrl c ctrl x

macOS:

cmd c cmd x

11- expandir o contraer selección

Windows:

Alt shift flecha derecha ó izquierda

macOS:

ctrl shift flecha derecha ó izquierda

12- Expandir entre etiquetas

Abrir command Palette:

Windows

ctrl shift p

macOS

cmd shift p

Buscar “balance” y seleccionar:

Emmet: Balance(outward)

Este comando por lo general no viene con un atajo de teclado preestablecido pero puedes crear uno.

Crear o modificar atajos de teclado

13- Comentarios

Comentar línea. Ubicar el cursor en cualquier parte de la línea y:

Windows:

ctrl }

macOS:

cmd shift 7 (cmd /)

Comentar bloque. Seleccionar bloque y:

Windows:

shift alt a

macOS

cmd shift A

14- Formatear documento o selección.

Este commando acomoda las incorrectas identaciones que tengamos en el código.

Abrir command Palette y buscar: Format

Vemos que aparece Format Document y Format Selection.

15- Agregar link css en el head del documento HTML

link:css Enter

16- Agregar meta viewport (responsive) en el head del documento HTML

meta:vp Enter

17- Contraer y descontraer Código

Contraer

Windows

Ctrl K Ctrl [

macOS

cmd k cmd 0 (cero)

Descontraer

Windows:

Ctrl K Ctrl ]

macOS

cmd k cmd j

18- Esconder panel lateral

Windows:

ctrl b

macOS

cmd b

19- Agregar Lorem ipsum.

Para crear texto de relleno escribimos

lorem Enter

Esto agrega 20 palabras. Si queremos personalizar la cantidad de palabras a lorem de agregamos el número de palabras

lorem5 Enter

Extensiones

css formatter

Permite formatear css

tag inserter

Permite insertar etiquetas de apertura y cierre sobre una selección

open in browser

Abre nuestro Código html en el navegador

Personalizar atajos.

Preferencias(engranaje ángulo inf izq) – keyboard Shortcuts Buscar el comando, seleccionarlo y Enter.

Presionar las teclas del atajo nuevo o el atajo a editar
