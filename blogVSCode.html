Visual Studio Code. 22 atajos y utilidades para simplificarte la vida.

15/5/20 por agustinpfs

1) Creación estructura Html:

! Enter ó html:5 Enter

2) Crear etiqueta con clase

etiqueta.nombreclase Enter

3) Duplicar línea o bloque.

Colocar cursor sobre la línea a duplicar o seleccionar el bloque de código a duplicar. alt shift flecha arriba ó abajo

4) Mover línea o bloque

Colocar cursor sobre la línea a mover o seleccionar el bloque de código a mover. alt flecha arriba ó abajo

5) Múltiple cursor

alt click

6) Múltiple cursor en columna

Windows: ctrl Alt flecha arriba o abajo OSX: cmd Alt flecha arriba o abajo

7) Seleccionar línea entera

Windows: ctrl L(minúscula) OSX: cmd L(minúscula)

8) Deseleccionar

Cuando seleccionamos varias líneas con cmd L es posible que nos pasemos. Para deseleccionar: Shift flecha arriba

9) Buscar repeticiones de una selección

cmd d

10) Copiar ó cortar línea entera

Sólo basta con colocar el cursor en cualquier parte de la línea Windows: ctrl c ctrl x OSX: cmd c cmd x

11) expandir o contraer selección

Windows: Alt shift flecha derecha ó izquierda OSX: ctrl shift flecha derecha ó izquierda

12) Expandir entre etiquetas

Abrir command Palette: osx cmd shift p win ctrl shift p y buscar “balance” y seleccionar: Emmet: Balance(outward) Este comando por lo general no viene con un atajo de teclado preestablecido pero puedes crear uno -> crear o modificar atajos de teclado

13) Comentarios

Comentar línea: Ubicar el cursor en cualquier parte de la línea win: ctrl } osx: cmd shift 7 (cmd /) Comentar bloque: Seleccionar bloque win: shift alt a osx cmd shift A

14) Formatear documento o selección.

Este commando acomoda las incorrectas identaciones que tengamos en el código. Abrir command Palette y buscar: Format Vemos que aparece Format Document y Format Selection.

15) Agregar link css en el head del documento HTML

link:css Enter 16) Agregar meta viewport (responsive) en el head del documento HTML meta:vp Enter 17) Contraer Código win Ctrl K Ctrl [ osx cmd k cmd 0 (cero) 18) Descontraer Código win: Ctrl K Ctrl ] osx cmd k cmd j 19) Esconder panel lateral win: ctrl b osx cmd b 20) Agregar Lorem ipsum.
