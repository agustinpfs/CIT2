¿Qué es Javascript?

11/7/20 por agustinpfs

Definición, variables, funciones, condicionales, operadores, arrays, loops, objetos y métodos, DOM. Ejemplos.

JavaScript

Javascript es un lenguaje de programación que pueden interpretar todos los navegadores web modernos.
Con Javascript conseguimos efectos especiales y la interactividad de un sitio web, es decir, dotar al usuario la capacidad de hacer cambios en una página web. Por ejemplo, botones para mostrar o ocultar contenido. Los pequeños programas que usamos para realizar estas acciones los llamamos scripts.

Variables

Una variable es un espacio de memoria al que le damos un nombre. En ese espacio podemos guardar diferentes datos, tales cómo números o texto.

nombre_variable1 = 5;
nombre_variable2 = "hola";

Colocar un valor a la variable se lo conoce como “asignar un valor”.
Estos valores, Javascript los clasifica por tipo de datos.
Los datos del ejemplo representan 2 de los principales tipos de datos.
Number representa números con los cuales podremos hacer operaciones.
String representa cadenas de texto, Deben escribirse entre comillas simples o dobles.

En estos ejemplos estamos asignando valores a variables no declaradas. Esto no es una buena práctica ya que son susceptibles a errores y hace que el código sea difícil de leer y mantener.

Para declarar una variable anteponemos al nombre de la variable las palabras const, let o var. Estas 3 palabras reservadas definen 3 tipos de variables.

const: Indica que el valor del dato no puede variar durante la ejecución del programa (no puede ser reasignada). Por ello cuando nos referimos a const lo hacemos como “constante” y no “variable”.

let y var son tipos variables que pueden variar durante la ejecución del programa (pueden ser reasignadas). Se diferencian según el alcance(scope) que afecte al código. Para mejor entendimiento, veremos estas diferencias en otro artículo ya que antes debemos comprender funciones y condicionales.

Funciones

Una función en JavaScript es un bloque de código que se ejecutará cuando invoquemos dicha función. Son útiles para rehusar código con diferentes valores que se pasan por parámetro.

Los parámetros son opcionales y sirven para pasar valores que serán utilizados por el código de la función.
Los argumentos son los valores que les envío a los parámetros.
Las sentencias son el código que se ejecutará.

Arrays(arreglos)

Es una colección de elementos que podemos asignar a una variable. Un array nos permite guardar datos de diferentes tipos, tales como números o cadenas de texto.
Podemos acceder a cualquiera de los elementos a través de sus índices numéricos y editarlos, borrarlos, insertar un nuevo elemento, etc. También los podremos recorrer mediante métodos de iteración.

Objetos y Métodos Javascript

Objetos JavaScript

Los objetos son unidades que contienen su propio código y datos para su propio auto-funcionamiento.
Podemos decir que son programas dentro de programas.
Generalmente encapsulan variables (propiedades como clave : valor) y funciones (métodos).
Los objetos se comunican entre ellos y nos permiten reutilizar código y hacer más fácil mantener nuestros programas.

Objetos y Métodos predeterminados de JavaScript

Javascript ya viene con objetos predeterminados (globales) y cada objeto con varias propiedades y métodos útiles para realizar nuestros programas o scripts.
Algunos de estos objetos predeterminados son:

  • String
  • Array
  • Math

String.

Incluye métodos como:

length: Devuelve la longitud de una cadena de texto.

var cadena = "hola";
var longitud = cadena.length;
console.log(longitud); //4

toUpperCase(): Devuelve la cadena en mayúscula.

var cadena = "hola";
var mayuscula = cadena.toUpperCase();
console.log(mayuscula); // HOLA

Array.

Incluye métodos como:

push(): agrega elementos al final del array.

let animales = ["gato", "perro", "caballo"]
console.log(animales); // [ 'gato', 'perro', 'caballo' ]

animales.push("vaca", "conejo")
console.log(animales); // [ 'gato', 'perro', 'caballo', 'vaca', 'conejo' ]

length: Devuelve la longitud de un arreglo.

var animales = ["gato", "perro", "caballo"];
var longitud = animales.length;
console.log(longitud); // 3

Math.

Permite realizar operaciones matemáticas.

Incluye métodos como:

pow(): devuelve el resultado de un número(1º argumento) elevado a una potencia(2º argumento).

console.log(Math.pow(5, 2)); // 25

random(): devuelve un número aleatorio entre 0 y 1

console.log(Math.random()); // 0.42499662557833595
